Tutorial: Using cURL API

You can do HTTP calls using cURL with the internal API.

All the parameters are documented in the Server API

Example code


API.curl({
  body: {
    url: 'http://your-url-here',
    method: 'GET'
  }
}, function(error, response) {
  console.log(response.body); // What you get back
});
