Adding Dropbox VFS support

HTTPS ONLY!

Dropbox support requires you to install an external repository: 

git clone https://github.com/dropbox/dropbox-js.git vendor/dropbox-js

# You may have to build dropbox.js manually before continuing

ln -sf vendor/dropbox-js/lib/dropbox.js dist/vendor/dropbox.js
ln -sf vendor/dropbox-js/lib/dropbox.min.map dist/vendor/dropbox.min.map
ln -sf vendor/dropbox-js/lib/dropbox.js dist-dev/vendor/dropbox.js
ln -sf vendor/dropbox-js/lib/dropbox.min.map dist-dev/vendor/dropbox.min.map

Simply create a config file to enable both API and VFS. Ex src/conf/903-dropbox.json: 

{
  "settings" : {
    "frontend" : {
      "Core" : {
        "Preloads": {
          "dropbox": {"type": "javascript", "src": "/vendor/dropbox.js"}
        },
        "VFS": {
          "Dropbox": {
            "Enabled": true,
            "ClientKey": "INSERT YOUR CLIENT ID HERE"
          }
        }
      }
    }
  }
}

Then run: 

grunt config
