Configure Your Google Developer Account

First off, you need a developer account at google (and a normal Google account).

Please note that some of the APIs only work with HTTPS. So I hightly recommend to set this up on your server

Step 1 - Set up account and project

Go here to make one. Then create a new project named "OS.js".

Step 2 - Set up credentials

Find the "Credentials" menu entry (left side). Create a new OAuth Client ID. The "Client ID" is what you will be using to configure support.

Step 3 - Enable API modules

Find the "APIs" menu entry (left side). Then click on the APIs you want to enable.

These are the modules that OS.js uses:

  • Contacts API
  • Gmail API
  • Google+ API
  • Drive API

Adding Google API and VFS support

Simply update the configuration: 

grunt config:set:client.GoogleAPI.ClientId:YOUR_CLIENT_ID
grunt config:set:client.VFS.GoogleDrive.Enabled:true
grunt config
