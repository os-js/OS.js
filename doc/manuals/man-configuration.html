Configuring and building

You can find all the configuration files in src/conf directory.

You can create your own files to override these. Just create (ex) 900-custom.json and place which settings there (example).

If you make any changes, issue this command to rebuild configuration files: 

grunt config

Building OS.js

OS.js has two build directories.

dist

This is the production build of OS.js. All files have been compiled into single files (and can be compressed) to increase loading speed. This will however make it harder to debug.

To build simply do: 

grunt all

dist-dev

This is what you should use when developing or testing changes.

You don't need to run any commands to update source files, since they are loaded directly.
