Adding Windows Live API and OneDrive VFS support

Simply create a config file to enable both API and VFS. Ex src/conf/902-wl.json: 

{
  "client" : {
    "WindowsLiveAPI": {
      "ClientId": "INSERT YOUR CLIENT ID HERE"
    },
    "VFS": {
      "OneDrive": {
        "Enabled": true
      }
    }
  }
}

Then run: 

grunt config
