Setting up Broadway

With Broadway you can run native GTK+ 3.x Linux applications in OS.js via WebSockets.

This is not yet ready for production: Github issue

Enable package

Edit src/packages/default/Broadway/package.json and set enabled to true

Build

dist

grunt config core packages manifest

dist-dev

grunt config manifest

Prepare X server

Run these commands on the server: 

cd Broadway/vendor/
npm install
xhost +
./broadway-server.sh

Usage

To use in OS.js just click the notification icon in panel and connect.
