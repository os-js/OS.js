Tutorial: Create your own application

Create from template

Simply run:

grunt create-package:MyName

This will create some source files for you with usable code. You'll have a basic running application with a Window.

Update manifest

The manifest holds information about your package, like name, description, icon and used resources. Update package.json to fit your application.

This is not required and can be done later

Example with all available entries: 

{
  // ClassName used in namespaces
  "className":  "ApplicationMyName",

  // If 'false' application will not be loaded or launchable (Optional)
  "enabled":    true,

  // Application description (optional)
  "description":"Application Long Description",

  // Application name
  "name":       "Application Name",

  // Application category (optional)
  "category":   "unknown",

  // Or an array of regext match MIME strings (Optional)
  "mime":       null,

  // Application icon (use ./my-icon.png for icon inside app folder)
  "icon":       "apps/file-manager.png",

  // Allow only one instance ? (Optional)
  "singular":   false,

  // Show a splash screen when loading application (Optional)
  "splash":     false,

  // Array of features to check for (Optional, uses Utils.getCompability())
  "compability":[],

  // A list of files to preload in order to run app
  "preload":    [
    {"src": "main.js", "type": "javascript"},
    {"src": "main.css", "type": "stylesheet"},
    {"src": "foo.js", "type": "javascript", combine: false} // Do not combine this script in 'dist'
  ],

  // For translations of name and description
  "names" : {
    "no_NO" : "Norwegian name"
  }
  "descriptions" : {
    "no_NO" : "Norwegian description"
  }
}

Install

Now run this command to add it to global manifest (make it visible in OS.js)

grunt manifest

And, if you're using dist (prodiction environment)

grunt packages
